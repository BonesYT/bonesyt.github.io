Score

Candy Crush engine

Version 1.0. Currently classic only.

i remade the engine from that game.

Click on 2 tiles to swap

Credits to King for the original series.

Formations

3 in row to clear; 4 in row for striped; T, L, + shape with 5 for wrapped, 5 in row for color bomb.

Swap combos

Striped+Striped: clear row and column Striped+Wrapped: clear 3 rows and columns Wrapped+Wrapped: 5x5 explosion twice ColorBomb+Normal: clear all of the same color ColorBomb+Special: replace all of the same color with special and triggers all ColorBomb+ColorBomb: clears the whole table
