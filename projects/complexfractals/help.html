Help for Complex Fractal Editor

Variables

c = starting point z = current point t = tick ti = time in ms re = c real im = c imaginary m = mouse pos in image mr = m real mi = m imaginary x = screen x pos y = screen y pos id = screen pixel ID mx = mouse x my = mouse y i = current iteration it = max Iterations jr = julia real ji = julia imaginary b = bailout value // for coloring speed = color density off = color gradient offset

Fractal formula object properties

Construct: { a: 4, b: 5 } // a and b are properties, 4 and 5 are their values. def = iteration formula onpixel = execute on pixel c = set c value z = set z value multi = skip iterations undo = reverse iteration bailout = bailout formula
