RGB Modifier Help

Want to know how the modes work? Read stuff below!

Values RGB

Separates every Red, Green, Blue channels in a pixel into 3 pixels. They will be shown in grayscale. (Alpha is ignored)

Values RGBA

Separates every Red, Green, Blue, Alpha channels in a pixel into 4 pixels. They will be shown in grayscale.

Values RG,BA

Separates every Red, Green, Blue, Alpha channels in a pixel into 2 pixels. Each of these pixels will have green and blue channels only. First pixel: Red, Green; Second: Blue, Alpha.

Binary 8px

Separates every pixel into 8. Every channel will be extended and outputed as binary (either 0 or 255), Making the image have 16 (9 visible) different colors. (Every pixel will be x4 wider and x2 taller.)

Binary 24px bits>channel

Separates every pixel into 24. Every channel will separate like 'Values RGB', and then separate into 8 bits. (Every pixel will be x8 wider and x3 taller.)

Binary 24px channel>bits

Separates every pixel into 24. Every RGB channel will split up into 8 pixels, then every part of each channel will be one of the channel bits. (Every pixel will be x8 wider and x3 taller.)

Binary 32px bits>channel

Separates every pixel into 32. Every channel will separate like 'Values RGBA', and then separate into 8 bits. (Every pixel will be x8 wider and x3 taller.)

Binary 32px channel>bits

Separates every pixel into 32. Every RGBA channel will split up into 8 pixels, then every part of each channel will be one of the channel bits. (Every pixel will be x8 wider and x4 taller.)
